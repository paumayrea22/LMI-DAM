# Lenguaje de Marcas e Intercambio de Información — 1.º DAM

Este repositorio recopila una selección de los proyectos y casos prácticos desarrollados en el módulo de **Lenguaje de Marcas (LMI)** en el IES Henri Matisse. El objetivo de este espacio es demostrar competencia en el almacenamiento, validación, transformación y presentación visual de información estructurada.

---

## 🛠️ Tecnologías y Competencias Clave

El contenido de este repositorio refleja el dominio de los estándares de la industria para el intercambio de información independiente de la plataforma:

* **Estructuración y Almacenamiento:** Diseño semántico de documentos utilizando esquemas dinámicos en formatos **XML** y **JSON**, asegurando la interoperabilidad de datos entre aplicaciones.
* **Consultas Avanzadas:** Uso y optimización de expresiones **XPath** para la navegación por la jerarquía de nodos, filtrado selectivo y extracción precisa de información en árboles XML complejos.
* **Maquetación y Diseño Frontend:** Creación de interfaces y páginas web visuales utilizando **HTML5** estructural y hojas de estilo **CSS3**, aplicando principios de adaptabilidad y código limpio.

---

## 📂 Organización del Código

El repositorio se organiza de forma clara para su evaluación:
* `/xml-json`: Ficheros de configuración y modelos de datos estructurados para transporte de información.
* `/xpath`: Scripts y ejercicios de extracción e indexación de nodos XML.
* `/web-interfaces`: Páginas web y maquetación estática con HTML y CSS.
